# Razorchess
Razorchess is an open source Chess game which allows the user to play against the Stockfish chess engine.

# Contributing 

## Prerequisites:
  - yarn
  - npm
  - nodejs

## Run Development Mode:

0. git clone https://github.com/tlewis11/razorchess.git
1. cd razorchess
2. yarn install
3. npm run dev
